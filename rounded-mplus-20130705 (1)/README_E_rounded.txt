Rounded M+ FONTS


'Rounded M+ FONTS' is a modification of 'M+ OUTLINE FONTS'
in rounded edges.

The license of this font is the same as 'M+ OUTLINE FONTS'.

This release is built with the source of
'M+ OUTLINE FONTS' on 2013-07-03T00:00:00+09:00.
